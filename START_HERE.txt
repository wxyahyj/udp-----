【我想在本地运行】
================================================================================

【方案 A: Python 版本 (推荐快速开始)】

  前置条件:
    • Python 3.8+
    • FFmpeg (在 PATH 中)
    • 依赖库

  安装:
    $ pip install -r requirements.txt

  运行:
    $ python udp_streamer.py -ip 192.168.1.100 -p 10000

  更多参数:
    $ python udp_streamer.py --help


【方案 B: C++ 版本 (最高性能)】

  前置条件:
    • Visual Studio 2022+
    • CMake 3.10+
    • FFmpeg 开发库

  编译:
    $ build.bat  # Windows
    # 或
    $ mkdir build && cd build && cmake .. && cmake --build .

  运行:
    $ build\Release\udp_streamer.exe -ip 192.168.1.100 -p 10000

【常见问题快速答案】
================================================================================

Q: 这个项目是什么？
A: 高性能 UDP 推流发射器，与 OBS 流媒体功能相同。
   支持硬件编码、低延迟、高帧率推流。

Q: 我应该从哪里开始？
A: 1. 看这个文件 (START_HERE.txt)
   2. 看 GITHUB_SETUP_SUMMARY.txt
   3. 按照指示部署到 GitHub

Q: 需要多长时间学会使用？
A: 5 分钟基础使用，15 分钟深入了解。

Q: 可以在哪些系统上运行？
A: Windows (x64/x86)、Linux、macOS、Docker。

Q: 性能如何？
A: 延迟 <50ms，帧率 60fps+，支持 4K 分辨率。

Q: 需要什么硬件？
A: 最低: 2GB RAM，推荐: GPU (NVIDIA/AMD)。

Q: 所有文件都完整吗？
A: 是的，共 33 个文件。运行 verify_project 脚本检查。

Q: 如何部署到 GitHub？
A: 3 个步骤，5 分钟。参考 GITHUB_SETUP_SUMMARY.txt。

Q: 如何获得帮助？
A: 查看相应的文档。快速问题看 QUICKSTART_CI_CD.md。
